# RV-CornerStone

RoboVigor 电控基石。

## 规范

- 永远采用最佳实践
- 使用配置文件调整参数
- 使用单独的user文件夹适配不同车种
- 使用utf-8
- 采用代码规范
- 采用FreeRTOS
- 采用[Gitflow Workflow](https://www.atlassian.com/git/tutorials/comparing-workflows/gitflow-workflow)作为版本管理
